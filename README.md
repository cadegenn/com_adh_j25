com_adh_j25
===========

Joomla! 2.5 component to manage people within an association

Features
--------
administration:
* dashboard
* complete list of people with filters (by chunks of name, first letter, city, country)
* extraction of mails and emails

configuration:
* association
* legal notice
* payment type
* email-alerts

site:
* registration page
